FAQ

Questions & answers.

Is Aether Radio really free?

Yes — completely free, open source, MIT licensed. No ads, no subscriptions, no accounts, no trackers. You can donate via Liberapay if you'd like to support development.

Where do the stations come from?

The community-driven radio-browser.info database — over 50,000 stations curated by volunteers worldwide. You can also add custom streams.

Which platforms are supported?

Android 6.0+, macOS (Intel & Apple Silicon), Windows 10/11, Linux (AppImage/.deb), and the web app in any modern browser.

Is there a browser version?

Yes — Aether Radio Live runs entirely in your browser with full feature parity. No install needed.

Does song recognition work offline?

Audio fingerprinting happens on-device, but matching requires an internet connection.

What data does Aether Radio collect?

None. Favorites, history and settings live only on your device.

Can I contribute?

Absolutely — open an issue, send a PR, or suggest a feature on GitHub.
